# ADR 0001: Adopt Base Repository Template

## Status
Accepted

## Context

A consistent repository structure improves maintainability,
discoverability, onboarding, and long-term evolution.

## Decision

Use a shared base repository template that defines:

- Documentation structure
- Contribution workflow
- Repository governance
- Formatting conventions

## Rationale

Standardization reduces cognitive load and prevents structural drift.

## Consequences

### Positive
- Faster project setup
- Predictable structure
- Easier collaboration

### Negative
- Some flexibility reduced
- Template maintenance required
